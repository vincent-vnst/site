Projet 1
Découverte du HTML et du CSS

Description du projet

Ce premier projet consiste à créer un mini-site en HTML et CSS. L’objectif est de comprendre la structure d’une page web, d’utiliser les balises sémantiques et d’appliquer les premières règles de style CSS.

Objectifs pédagogiques

  • Découvrir la structure d’un document HTML5.
  • Apprendre à organiser un site web avec plusieurs pages liées.
  • Utiliser le CSS pour la mise en forme du contenu.
  • Publier le site sur GitHub Pages.

Compétences visées

  • Structurer et organiser un site web en utilisant le HTML sémantique et les bonnes pratiques de codage.
  • Mettre en forme et personnaliser l’apparence d’un site avec le CSS (Flexbox, Grid, couleurs, polices, effets).
  • Créer un portfolio complet avec une navigation cohérente.
  • Intégrer des contenus multimédias accessibles et esthétiques.
  • Adapter un site aux différents écrans grâce au responsive design et aux media queries.
  • Déployer un site en ligne via GitHub Pages et le valider (W3C, Lighthouse).
  • Présenter et valoriser son travail avec une identité graphique cohérente.

Résultat obtenu

Le projet final se compose d’une page d’accueil, d’une galerie de projets et d'une page par projet. Le site est responsive et conforme aux standards du W3C.
